Research Projects

Development of Smart and Sustainable Agriculture using IoT

This project aimed to develop an IoT-based smart agriculture system that can efficiently monitor and control the environment of crops using various sensors and actuators. It involved designing and implementing a custom hardware and software solution to gather real-time data on temperature, humidity, soil moisture, and light intensity. The data was then analyzed and used to automate the irrigation and fertilization process, resulting in improved crop yields and reduced water consumption.

  • Duration: January 2018 - December 2019
  • Role: Project Lead
  • Skills: IoT, Embedded Systems, Sensor Networks, Data Analysis
